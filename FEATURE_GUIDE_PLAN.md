# 세 가지 방법으로 즐기는 AI 그림 놀이 - 이용 가이드 기획안

사용자가 메인 페이지에서 "AI 그림 놀이" 기능을 더 쉽고 재미있게 이해할 수 있도록 돕는 시각적 가이드 구성안입니다.

## 1. 마법 그림판 (Magic Paint)
- **설명**: "AI가 내 필력을 마법처럼 재해석합니다."
- **가이드 포인트**: 
    - 캔버스에 자유롭게 상상력을 발휘하세요.
    - [카툰 / 동화 / 마법 3D] 중 원하는 스타일을 선택하세요.
    - AI 완성 버튼을 누르면 상상이 현실로 변합니다.
- **핵심 팁**: "명확한 선과 구도를 그릴수록 더 완벽한 AI 결과물이 탄생합니다."

## 2. 거울 그림판 (Mirror Paint / Decalcomania)
- **설명**: "반만 그려도 완성되는 대칭 예술!"
- **가이드 포인트**: 
    - 캔버스 왼쪽에만 그림을 그려보세요.
    - 실시간으로 오른쪽에도 똑같은 그림이 그려집니다.
    - 데칼코마니 같은 신비로운 대칭 작품을 감상하세요.
- **핵심 팁**: "중앙선을 넘나들며 그리면 독특한 기하학적 패턴을 만들 수 있습니다."

## 3. 시간초 그림판 (Timer Paint / Time Attack)
- **설명**: "60초의 긴장감, 제시어를 빠르게 그려보세요!"
- **가이드 포인트**: 
    - 상단에 표시된 '오늘의 제시어'를 확인하세요.
    - 60초가 다 되기 전에 특징을 잡아 그려야 합니다.
    - AI가 정답을 맞히면 성공! 친구와 점수 대결을 해보세요.
- **핵심 팁**: "복잡한 묘사보다 사물의 특징(예: 코끼리의 코, 고양이의 귀)을 살려 그리는 것이 정답의 비결입니다."

---

## 구현 제안 (UI/UX)
- 각 섹션에 **[이용 방법 ?]** 버튼 추가.
- 클릭 시 해당 놀이의 플레이 화면 녹화(GIF) 또는 간단한 3단계 스크린샷 팝업 노출.
- '카드 뒤집기(Flip Card)' 애니메이션을 활용한 뒷면 설명 배치 제안.
